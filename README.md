# 开封围城记

## 简介

这是一部历史穿越小说，讲述了现代计算机系研究生朱文磊意外穿越到明朝崇祯年间，成为周王世子的故事。

## 故事背景

- **时间**: 明朝崇祯十四年（1641年）
- **地点**: 河南开封府
- **历史背景**: 李自成起义军攻陷洛阳后，主力突袭河南省会开封，拉开了开封围城战的序幕

## 主要情节

### 第1章 跳闸！下河游泳混穿世子
现代研究生朱文磊在开封老城区的酷暑中，因电路跳闸去龙亭湖游泳，意外触碰到湖底的神秘异石，穿越到明朝崇祯十五年五月的开封，与周王世子朱文磊的灵魂在暴雨战火中完成融合。

### 第2章 穿越了不算差
朱文磊醒来后发现自己成为了大明周王世子，身处崇祯十四年二月的开封府。面对李自成大军的围城，他决心改写开封被黄河大水淹没的历史悲剧。

### 第3章 收买人心
朱文磊利用前世的学生会经验，通过宽恕失职的侍卫并赏赐银两，开始收买人心，建立自己的核心班底，为后续的改命之路奠定基础。

# 人物设定

## 主要人物

### 朱文磊（主角）
- **身份**: 现代计算机系研究生 → 明朝周王世子
- **背景**: 
  - 现代: 开封土生土长，研二计算机系学生，家境不富裕
  - 古代: 大明周王世子，宗室勋贵
- **性格**: 
  - 现代: 勤奋、有历史知识、学生会主席经验
  - 古代: 原主骄纵，穿越后冷静、善于权谋
- **能力**: 
  - 历史大局观
  - 学生会管理经验（御下之道）
  - 工科知识（可用于改良火器）
- **目标**: 改写开封被黄河大水淹没的历史悲剧

### 朱恭枵（老周王）
- **身份**: 大明周王
- **性格**: 
  - 在大难临头时没有怯懦
  - 散尽王府家财死守城池
  - 关爱儿子
- **历史评价**: 大明宗室中最靠谱、最硬气的一支
- **结局**: 历史上撤往南方，得以寿终正寝

## 历史人物

### 李自成
- **身份**: 闯王，起义军首领
- **背景**: 银川驿驿卒出身
- **特点**: 
  - 天生的统帅
  - 擅长特种信息战与通信遮蔽
  - 利用"迎闯王，不纳粮"口号吸引百万饥民
- **战绩**: 
  - 崇祯十一年梓潼战役全军覆没
  - 崇祯十三年突入河南
  - 崇祯十四年攻陷洛阳，烹杀福王
  - 三次围攻开封

### 刘宗敏
- **身份**: 李自成麾下头号悍将
- **战绩**: 
  - 率三千精骑伪装明军企图诈开大梁门
  - 诈城失败后驱赶百姓蚁附登城

### 高名衡
- **身份**: 河南巡按
- **特点**: 
  - 眼疾手快，当机立断
  - 铁血手腕
- **战绩**: 
  - 死守大梁门，识破诈城计谋
  - 临时用王府银子招募市民死守

### 黄澍
- **身份**: 推官
- **战绩**: 与高名衡一起死守城门

### 陈永福
- **身份**: 开封总兵官
- **特点**: 开封城防主力
- **处境**: 
  - 带着城内唯一常备精锐西进驰援洛阳
  - 洛阳失陷后拼命运回
  - 在第一围中回防射瞎李自成左眼

### 朱常洵（福王）
- **身份**: 万历皇帝最宠爱的儿子，大明最富庶的藩王
- **性格**: 吝啬，宁死不肯拿出一两银子犒赏守城将士
- **结局**: 洛阳城破后被李自成烹杀，史称"福禄宴"

## 原创人物

### 王府仪卫司侍卫
- **身份**: 周王府仪卫司贴身侍卫
- **处境**: 
  - 前日没能护住世子导致其坠马昏迷
  - 按王府规矩，世子不治则被乱棍打死、全家充军
- **结局**: 被朱文磊宽恕并赏赐银十两，成为其铁杆班底

# 03_world_setting.md

## 地理环境
- **开封府**: 中州首府，人口百万。
- **"城摞城"奇观**: 周王府、金皇宫、北宋大内叠压。
- **大梁门**: 西门，李自成进攻的重点方向，城墙高大。
- **龙亭湖**: 周王府后花园，深不可测。

## 社会制度
- 宗藩体制，明末社会矛盾尖锐。
- 地方官府与王府关系微妙，战时周王府地位超然。
- 城内保甲制度，市民、太学生组成社兵。

## 🏛️ 军事体系与指挥架构（全新并入）
### 1. 战时文武双轨指挥链
*   **文官统筹（战略、后勤、戒严情报）**：
  *   **河南巡按御史高名衡**（字平仲，号鹭矶，正七品衔，当前全城政治与动员最高调度，内政极强，军事拉胯，立志做睢阳齿抱死守之心）。
  *   **开封府推官黄澍**（字若霖，正七品，总管战时全城特务防谍、抓捕闯军内奸，为人精明阴鸷，手段狠辣爱出毒计）。
  *   **祥符县令王燮**（正七品，组织基层保甲丁壮、调配城内基础钱粮发放）。
*   **武将防御（战术指挥、正面攻防）**：
  *   **河南副总兵陈永福**（从二品差遣职，防守大师。出身边军，治军严整，但作风保守，极怕承担战败被文官弹劾的政治风险）。
  *   **游击陈德**（陈永福之子，守城一线的青年悍将）。

### 2. 地方驻军：河南都司卫所
*   **核心番号**：开封卫、河南卫、彰德卫。
*   **现状**：严重空壳化，吃空饷普遍。名籍在城头多流于番号防线，但保留了大量的武备仓库，并留有王铁锤等世袭操炮老兵。
*   **卫所武职职级序列（从高到低）**：指挥使（正三品） -> 指挥同知（从三品） -> 指挥佥事（正四品） -> 正千户（正五品）/副千户（从五品，管1120人） -> 百户（正六品，管112人） -> 总旗（正七品，50人） -> 小旗（从七品，10人）。

### 3. 驿传信息战
*   李自成切断陈留、朱仙镇驿传，制造信息屏障。将开封化作一座与外界彻底断绝往来的“信息黑洞”。

## 👑 周王府独立军事力量（全新并入）
### 1. 仪卫司（常备精锐保安）
*   实员 500-800 人（由仪卫正/副统领，如高应登、刘长庚）。平日掌宫门站岗、世子仪仗。身披王府内库精甲，装备极度奢华。虽然没经历过大规模野战，但单兵贴身肉搏能力远超普通明军。在主角施恩后，演变为主角最坚实的私人死士亲军。

### 2. 王府私属家丁
*   数百至千人（视王府散金力度而扩）。由老周王用重金私下豢养的绿林好汉或破产健卒组成。只认王命，不尊兵部。配有王府内库私藏的优质黑火药、连发强弩等特制精锐火器。

### 3. 宗室募兵与自卫联防
*   **社兵（王府出资）**：老周王散尽家财、计日散金临时在城内招募的几万市井青壮与太学生，缺乏组织。
*   **联防队**：由开封城内数千名中下层周藩宗室临时编组的自卫队，负责协防内库及特定王府路段。

## 💻 程序员世子的军事破局思维（全新并入）
1.  **风险对冲（释放武将潜能）**：陈永福因保守不敢冒险，高名衡军事拉胯但抱死志。主角作为超然世子，可通过独自承担政治风险来给陈永福彻底解绑，激活其“防守大师”的最大军事潜能。
2.  **基干模块化重组（组织力降维）**：利用**宗廷辅**（周王府长史）的行政统筹权，将仪卫司内身披精甲、懂战阵的精锐下放。充当几万乱哄哄的保甲市民的百户、总旗（基层连长排长），用现代分工管理强行重塑民兵战力。

## 经济状况
- 周王富庶，愿意散财守城。
- 与洛阳福王的吝啬形成鲜明对比。

## 军事装备
- 守城火器：红夷大炮、弗朗机、大将军炮。
- 城防守具：震天雷、神枪、挨牌、盾车、万人敌。
- 滚油、金汁、礌石、滚木。
- **主角黑科技（新增）**：湿式炮刷（Wet Swab，利用粗棉布、羊毛与猪鬃制作，清理炮膛残渣余火，防止连续发射导致的“炸膛”隐患，大幅提升大炮射频）。

## 时代背景
- 崇祯末年，中原大旱，流民遍地。
- 李自成罗汝才势大，杨嗣昌十面张网破产。
- 松锦大战爆发，关外满清八旗步步紧逼。
